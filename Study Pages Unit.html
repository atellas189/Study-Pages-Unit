Elements

Web Table
HTML structure that consists of multiple table rows and 1 or more columns
Table Cells
Each table row box
Header Cells
- Contains content placed at the top of a column or beginning of a row
- By default, displays text in bold and centers text horizontally
Data Cells
- Contains content within within columns or rows
- By default, displays text as unformatted text and is aligned to the left within the cell
Spanning Cells
Single cell that occupies more than one cell row and/or column
Caption
The content contained within the caption
thead
Rows that belong to the table head
tfoot
Rows that belong to the table footer
tbody
Rows that belong to the table body
CSS border property
Add borders to any part of a web table
rows
content from right to left
columns
content from top to bottom
td
table definition
th
table header, by default, content centers and bolds
tr
table row
Collased Border Model
Borders from adjacent elements are merged together to form a single border
colgroup
A column that becomes more than 1 columns
