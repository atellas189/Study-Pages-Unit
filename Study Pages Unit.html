Web Table
HTML structure that consists of multiple table rows and 1 or more columns
Table Cells
Each table row box
Header Cells
- Contains content placed at the top of a column or beginning of a row
- By default, displays text in bold and centers text horizontally
Data Cells
- Contains content within within columns or rows
- By default, displays text as unformatted text and is aligned to the left within the cell
Border-collapse
Separate or collapsed borders that apply to table cells
Border-spacing
Spacing between borders
Spanning Cells
Single cell that occupies more than one cell row and/or column
